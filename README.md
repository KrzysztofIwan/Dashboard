# Dashboard

## Opis projektu
DashBoard prezentuje informacje na temat wyścigów F1 z lat 1950 do 2024.

DataSet został pobrany z [Kaggle](https://www.kaggle.com/datasets/rohanrao/formula-1-world-championship-1950-2020?resource=download)

## Użyte technologie
* Python 3.13.5 - język bazowy
* streamlit 1.56.0 - forntend aplikacji
* pandas 2.3.3 - przygotowanie i obróbka danych

## Uruchomienie
Zainstalowanie wszystkich potrzebnych bibliotek 
```bash
pip install -r requirements.txt
```

Uruchomienie projektu z terminala 
```bash
streamlit run main.py
```
